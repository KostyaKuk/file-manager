### Basic Navigation
- **`up`** - Go up one level from current directory
- **`cd <path>`** - Change to specified directory (supports relative and absolute paths)
- **`ls`** - List directory contents in a formatted table

### File Operations
- **`cat <filename>`** - Read and display file content with syntax highlighting
- **`add <filename>`** - Create a new empty file
- **`rn <oldname> <newname>`** - Rename a file (content remains unchanged)
- **`cp <source> <target>`** - Copy file to target directory with same filename
- **`mv <source> <target>`** - Move file to target directory with same filename
- **`rm <filename>`** - Delete a file

### Directory Operations
- **`mkdir <dirname>`** - Create a new directory

### System
- **`.exit`** - Exit the file manager
- **`--username=YourName`** - Start with personalized greetings
